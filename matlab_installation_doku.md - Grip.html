Doku für Matlab Offline installation

1. MobaXterm so einstellen, dass auch mit Sudo ./install funktioniert:

In Terminal eingeben.

    sudo xauth add $(xauth -f ~jan/.Xauthority list|tail -1)

https://blog.mobatek.net/post/how-to-keep-X11-display-after-su-or-sudo/

2. Zuerst Matlab für Windows Version runterladen.

https://de.mathworks.com/downloads/web_downloads/

Nach dem entpacken auf setup.exe klicken.
Recht oben auf Matlab download only runterscrollen und wählen.

Es sollten mindesten 10GB groß sein.

Nach dem runterladen die ganze Datei auf die Lokale Home Ornder der Server verschieben/kopieren.

Danach im Matlab Ordner folgenden Befehl eingeben.

	sudo unzip -X -K matlab_R2020b_glnxa64.zip

3. Lizenz Aktivieren und die Keys holen

Hier im Link auf Install und Activate klicken:
https://de.mathworks.com/licensecenter/licenses/40871335/6757286/products

Danach auf den Link von "Activate to Retrieve License File" im rechts im Fenster klicken.

Nach dem klicken auf auf "Activate auf Computer" erscheint folgendes Fenster:

Operation System: Linux
Host ID : Die Mac Adresse der Server eingeben.
Für Host ID folgendes eintippen.

	ip addr | grep ether

Computer-Login Name: beispiel dein Username für den Server bst: Jan

Activation Label: s-csb-gpu - name für den Rechenr

Danach sollte die Fileinstallation Key und File zur verfügung stehen.
Bei der Frage ob Matlab schon installiert ist auf nein klicken.

Die File runterladen und den Key speichern.

4. Matlab Installieren.

Im Ornder von Matlab/

	sudo ./install

Danach Sollte Matlab Installations Fenster erscheinen.

Hier zustimmen.

Hier werden nach den Key und File gefragt, welche wir vorher runtergeladen haben.

Danach kommt die Installation.
